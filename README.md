# Instalacion_de_Matlab

El objetivo de este repositorio es facilitar la instalación de Matlab en Linux

## Iniciar sesión en MathWorks

[Iniciar sesión](https://login.mathworks.com/embedded-login/landing.html?cid=getmatlab&s_tid=gn_getml)

En caso de no tener sesión se debe registrar usando el correo estudiantil, puesto que el TEC tiene las credenciales activas.

## Instalar Matlab y Requisitos

Una vez estando en el menu principal, se selecciona la opción de "instalar Matlab", se abrira una nueva ventana e inmediatamente reconoce el sistema operativo, por lo cual unicamente se le da a "Descargar para Linux".

[Iniciar sesión](https://login.mathworks.com/embedded-login/landing.html?cid=getmatlab&s_tid=gn_getml)

Luego, extraer el archivo .zip que se descargo (puede extraerlo en la ubicación de su preferencia).

Abra la terminal estando en el folder extraido y ejecute el comando

sudo ./install 

A continuación, vuelva a iniciar sesión en la ventana emergente, acepte todos los acuerdos, seleccione la ubicación de instalación de su preferencia (perfectamente puede dejar la que viene por defecto) y colocamos los siguientes checks

- MATLAB
- Simulink
- Control System Toolbox
- Embedded Coder

[Iniciar sesión](https://login.mathworks.com/embedded-login/landing.html?cid=getmatlab&s_tid=gn_getml)
[Iniciar sesión](https://login.mathworks.com/embedded-login/landing.html?cid=getmatlab&s_tid=gn_getml)
[Iniciar sesión](https://login.mathworks.com/embedded-login/landing.html?cid=getmatlab&s_tid=gn_getml)

Aparecera una ventana emergente indicando que MATLAB Coder requiere añadir un par de cosas, presione "Add".

Recomendación: seleccionar "Create symbolic links to MATLAB scripts in: " para qque cree un acceso directo.

Finalmente haga click en "Begin Install".

## Duración

Se instalará aproximadamente 8.73 GB, dependiendo de la conexión a internet que tengan puede tardar
